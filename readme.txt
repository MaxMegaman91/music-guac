How to Use aYT_downloader

1) Login with login and password
2) Call for input mode
    - txt means it reads the.txt and downloads all inside it (format info inside .txt)
    - mp3 means it requires a format to install an mp3
    - emp3 means it requires separate inputs to install an mp3 
    - link means it requires a link and quality to install accordingly (press <enter>)
3) Done!
